# Trading
Tools created for working with trading data i.e. downloading data for ML tasks, specialized indicators

# Finds highs and lows script (find_highs_lows.py)

This script aims to identify highs and lows in noisy OHLC (Open-High-Low-Close) data using statistical analysis and visualization techniques. The script processes the data, calculates moving averages, identifies trends and inflection points, and plots the results with visual markers.

## Requirements

- Python 3.7+
- pandas
- numpy
- mplfinance
- logging
